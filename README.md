# 코드잇 영상 댓글 분석 웹자료

코드잇 유튜브 영상 4개의 댓글 반응을 정리한 정적 웹자료입니다. 영상별 썸네일 링크, 댓글 유형 분포, 키워드, 영상별 분석, 댓글 원문 아카이브, 콘텐츠 기획 제안을 포함합니다.

## 구성

- `index.html`: 웹자료 메인 페이지
- `styles.css`, `components.css`: 화면 스타일
- `app.js`: 데이터 로딩, 필터, 정렬, 렌더링 로직
- `data/*.json`: 영상, 댓글, 유형 데이터
- `data-inline.js`: 파일 직접 열기용 내장 데이터
- `analysis.md`: 웹자료 구성안과 분석 기준

## 확인 방법

`index.html` 파일을 브라우저에서 직접 열거나, 정적 서버로 실행할 수 있습니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://127.0.0.1:4173/`을 엽니다.
